# Kubernetes

The RomM team doesn't publish or formally support a first-party Helm chart or manifest set. For now, deploy via a community-maintained chart.

<!-- prettier-ignore -->
!!! warning "Work in progress"
    First-party Kubernetes guidance is being rewritten. Until it lands, treat this page as a pointer to community resources plus a short list of known gotchas. If you have a deployment worth contributing back, drop into the `#kubernetes` channel on the [Discord](https://discord.gg/P5HtHnhUDH).

Find common Kubernetes-specific issues in [Kubernetes Troubleshooting](../troubleshooting/kubernetes.md).

## Community charts

- **[ArtifactHub](https://artifacthub.io/packages/search?ts_query_web=romm)**: search for `romm` and pick a chart with active maintenance and recent releases.
- **[Discord](https://discord.gg/P5HtHnhUDH)**: the `#kubernetes` channel is a great place to ask which chart people are actually running.
